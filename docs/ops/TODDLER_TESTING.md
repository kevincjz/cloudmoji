# Toddler Testing Protocol

## Purpose
Structured observation sessions to validate H1 (engagement), H9 (speech imitation), and H10 (repetition patterns). This is the single most important research activity in the entire project.

## Setup

### Equipment
- iPad or iPhone with Cloudmoji loaded (full screen, "Add to Home Screen")
- Phone camera to record the child's face/reactions (mount or second person holding)
- Screen recording ON (iOS: Settings → Control Center → Screen Recording)
- Stopwatch (use a second phone)
- Printed observation sheet (template below)
- Quiet-ish environment (home, not playground)

### Participants
- 5 toddlers aged 18-36 months
- Must include Cloud (Kevin's son, 27 months)
- Ideally mix of: ages, genders, bilingual vs monolingual homes
- Recruit from: Kevin's friends, CrossFit Mobilus parents, PQ's mum network

### Pre-Session
1. Device volume at 70-80% (loud enough to hear clearly, not startling)
2. App open to "All" category, language set to English
3. Typing row empty
4. Brief explanation to parent: "Just let them explore. Don't show them what to do. If they ask you something, respond naturally, but don't guide their tapping."

## Session Protocol

### Phase 1: Free Exploration (0:00 – 5:00)
- Hand device to child
- Start stopwatch
- DO NOT demonstrate, point, or guide
- Observe and note:
  - [ ] Time to first tap: _____ seconds
  - [ ] First emoji tapped: _____
  - [ ] Does the child react to the spoken word? How? (looks up, smiles, repeats, ignores)
  - [ ] Tap rate in first minute: _____ taps
  - [ ] Does the child understand the typing row? (looks at it, taps emojis in it)

### Phase 2: Sustained Engagement (5:00 – 10:00)
If the child is still engaged at 5 minutes:
- Continue observing
- Note:
  - [ ] Does tap rate increase, decrease, or stay steady?
  - [ ] Does the child explore categories? Which?
  - [ ] Does the child develop favorites (tapping same emoji 3+ times)?
  - [ ] Any vocalization? Which words?
  - [ ] Does the child show the screen to parent/others?

### Phase 3: Natural End
- DO NOT stop the session — let the child decide when they're done
- Note:
  - [ ] Total session time: _____
  - [ ] How did the session end? (walked away / asked for something else / parent intervened / still going at 15 min and parent stopped)
  - [ ] Total taps (approximate or from event log): _____
  - [ ] Unique emojis (from event log): _____

### Phase 4: Parent Debrief (After Child Moves On)
Ask the parent:
1. "What did you notice?"
2. "Did anything surprise you?"
3. "Did [child] say any words?"
4. "Would you want to use this again tomorrow?"
5. "What would make it better?"

Record answers. These qualitative insights are as valuable as the numbers.

### H1 Verdict
- If 4/5 kids engaged 5+ min: **VALIDATED** → proceed with confidence
- If 2-3/5 kids engaged 5+ min: **WEAK** → look at what differed for engaged vs disengaged kids
- If 0-1/5 kids engaged 5+ min: **INVALIDATED** → test with older kids (4-6) or reconsider core mechanic

## Follow-Up (24-48 Hours Later)
Text each parent: "Has [child] asked for the emoji app again?"
- If yes: extremely strong signal
- If no but parent offers it and child engages: good signal
- If no and child isn't interested: weak signal
