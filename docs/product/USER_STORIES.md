# User Stories — Prioritized

## Phase 1: MVP

### Must Have (P0)
- [ ] **US-001**: As a toddler, I can tap an emoji and hear the word spoken aloud, so I learn what things are called
- [ ] **US-002**: As a toddler, I can see the emojis I've tapped in a row at the top, so I can see my "sentence"
- [ ] **US-003**: As a parent, I can switch the language so my child hears words in our home language(s)
- [ ] **US-004**: As a toddler, I can tap an emoji in my typing row to hear it again, so I can practice
- [ ] **US-005**: As a parent, I can filter emojis by category so my child focuses on a topic (animals, food, etc.)
- [ ] **US-006**: As a toddler, I can tap the replay button to hear all my emojis spoken in sequence
- [ ] **US-007**: As a parent, I can clear the typing row when it gets too long
- [ ] **US-008**: As a toddler, I see a word label briefly appear when I tap, reinforcing the word visually

### Should Have (P1)
- [ ] **US-009**: As a parent, my child's preferred language persists between app sessions
- [ ] **US-010**: As a toddler, I see fun bounce animations when I tap emojis, making it satisfying
- [ ] **US-011**: As a parent, the app works without internet so I can use it on planes/car rides
- [ ] **US-012**: As a parent, I can see a "How to use" tooltip on first launch

## Phase 2: Delight

### Must Have (P0)
- [ ] **US-013**: As a parent, I can see how many words my child has practiced this week
- [ ] **US-014**: As a toddler, I see confetti/stars when I hit word milestones (10, 25, 50, 100)
- [ ] **US-015**: As a parent, I can record my child saying a word and play it back
- [ ] **US-016**: As a parent, I receive a daily "Word of the Day" notification with a new emoji

### Should Have (P1)
- [ ] **US-017**: As a parent, I can share a video clip of my child's emoji typing + speech to WhatsApp
- [ ] **US-018**: As a toddler, I feel haptic feedback when I tap emojis (satisfying tactile response)
- [ ] **US-019**: As a parent, I can set a daily time limit for app usage
- [ ] **US-020**: As a parent, I can create multiple child profiles

## Phase 3: Intelligence

### Must Have (P0)
- [ ] **US-021**: As a toddler, I see emojis I haven't practiced recently surfaced more prominently
- [ ] **US-022**: As a toddler, I can build simple sentences by tapping sequences (🐶 + 🍎 → "Dog eats apple")
- [ ] **US-023**: As a parent, I can see speech recognition feedback on my child's pronunciation

### Should Have (P1)
- [ ] **US-024**: As a parent, I can add custom "emojis" using photos from my camera roll
- [ ] **US-025**: As a toddler, I can play a matching game (hear word → tap correct emoji)

## Phase 4: Platform

- [ ] **US-026**: As a teacher, I can assign emoji sets to my class
- [ ] **US-027**: As a teacher, I can see class-wide progress on vocabulary
- [ ] **US-028**: As a community member, I can contribute a new language pack
- [ ] **US-029**: As a developer, I can access the emoji/TTS API for my own app
